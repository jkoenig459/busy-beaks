# Busy Beaks

A "comfy" **city builder** made for **Comfy Jam: Winter 2026** (theme: *transitions*), where you guide a small colony of penguins rebuilding their village.

🏆 **Placed 7th out of 88 entries** — [Comfy Jam: Winter 2026](https://itch.io/jam/comfy-jam-winter-2026)
🎮 **Play it on itch.io:** https://cecixp.itch.io/busy-beaks

## Overview

- **Engine:** Unity (Universal Render Pipeline)
- **Language:** C#
- **Genre:** City builder / colony management (RTS-style)
- **Made for:** Comfy Jam: Winter 2026

## What I built

- Colony management and city-building systems in C#
- Resource gathering and building placement logic
- UI built with Unity UI Toolkit
- Custom outline shader and URP setup
- Player input handling with Unity's Input System

## Skills demonstrated

C# · Unity · UI design · RTS / city-management systems · complex gameplay logic

## Status

Built during a game jam, so some upgrades and building types were left unfinished due to time constraints. Planned next steps: finishing those systems and adding more resources to collect so the colony always has something to do.
